About

I'm a junior software developer based in Scotland, building full-stack applications with Python and React. My first major project — an AI-powered staff rostering system — taught me everything from constraint-based optimisation with Google OR-Tools to designing role-based dashboards with FastAPI and PostgreSQL.

I'm drawn to the full breadth of software engineering: backend APIs, frontend interfaces, deployment pipelines, and the AI layer that ties it all together. I learn by building, and I write about what I discover along the way so other developers starting out can benefit too.

When I'm not coding you'll find me exploring new frameworks, tinkering with deployment setups, or reading up on the latest in machine learning and optimisation. I'm actively looking for opportunities to grow as a developer and contribute to meaningful projects.
